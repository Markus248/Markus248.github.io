Projektseite zum Modul HTML / CSS


Eine kurze Erläuterung


Dies ist meine Projektseite, die im Rahmen meiner Umschulung zum Fachinformatiker in der Fachrichtung Anwendungsentwicklung in dem Modul HTML und CSS entstanden ist. In diesem Modul haben wir wesentliche Basistechniken im Umgang mit HTML und CSS kennen gelernt. Die Projektseite dient der Einübung der besprochenen Theorieteile und bildet im Wesentlichen alle besprochenen Inhalte ab. Lediglich der Sandwich Button wurde nicht im Rahmen des Modules gelehrt, sondern ist aus eigenen Recherchen entstanden. Ich verwende im Folgenden ausschließlich Dummy Bilder, Dummy Text und Screenshots, da der Fokus auf den Technologien, nicht auf dem Inhalt liegt.
